##审核佣金-拒绝

------------

###请求方式：
> POST

###请求地址：
> http://platform.360bao.com/api/commission/approval/reject

###授权：
> 需要授权，在request 的Header中加入
"Authorization":"Bearer "+jwt


###上行数据：
```
{
    "commissionNoList":["123","456"],
    "message":"消息"
}
```

###下行数据：
```
{
    "message":"",
    "result":true
}
```
